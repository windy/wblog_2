# Changelog

All notable changes to WBlog will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-01-20

### Added
- Rails 7 upgrade with modern features
- Hotwire support for dynamic interactions
- ESBuild for modern JavaScript bundling
- Stimulus controllers for enhanced interactivity
- Improved mobile responsiveness
- Better SEO optimization

### Changed
- Complete UI/UX overhaul
- Upgraded all dependencies to latest versions
- Improved markdown rendering
- Enhanced code syntax highlighting
- Optimized database queries
- Modernized asset pipeline

### Fixed
- Mobile display issues
- Comment submission errors
- Image upload reliability
- Search functionality
- Performance bottlenecks
- Security vulnerabilities

## [1.1.0] - 2020-06-15

### Added
- Multi-language support
- Image lazy loading
- Comment notifications
- Social media sharing
- RSS feed support

### Changed
- Improved comment system
- Enhanced admin dashboard
- Better mobile navigation
- Optimized image processing

### Fixed
- Cross-browser compatibility issues
- Text encoding problems
- Date formatting inconsistencies

## [1.0.0] - 2016-04-20

### Added
- Basic blog functionality
- Comment system
- Admin dashboard
- Post management
- Label system
- User authentication
- Image uploads
- Markdown support
- Basic search functionality
- Mobile-friendly design

### Notes
For detailed technical changes and migration guides, please refer to the documentation.
